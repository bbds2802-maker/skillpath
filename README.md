# SkillPath

AI-powered skill learning planner. Zero distraction.

## Pages
- [Website](https://bbds2802-maker.github.io/skillpath/web.html) - Desktop learning planner with full calendar view
- [App](https://bbds2802-maker.github.io/skillpath/app.html) - Mobile app version with bottom navigation

## How it works
1. Enter a skill you want to learn
2. Set your schedule preferences
3. Get a personalised, progressive lesson plan with curated video links
4. Track sessions on a real calendar

Built with vanilla HTML/CSS/JS.
